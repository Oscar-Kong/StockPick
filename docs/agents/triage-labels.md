# Triage Labels

Matt Pocock engineering skills use five canonical triage roles. This file maps those roles to GitHub label strings for StockPick.

| Role in skills | Label in GitHub | Meaning |
|----------------|-----------------|---------|
| `needs-triage` | `needs-triage` | Maintainer needs to evaluate this issue |
| `needs-info` | `needs-info` | Waiting on reporter for more information |
| `ready-for-agent` | `ready-for-agent` | Fully specified, ready for an AFK agent |
| `ready-for-human` | `ready-for-human` | Requires human implementation or judgment |
| `wontfix` | `wontfix` | Will not be actioned |

Create these labels in the GitHub repo settings if they do not exist yet. Edit the right-hand column if the project adopts different names.

When a skill mentions a role (e.g. "apply the AFK-ready triage label"), use the corresponding label string from this table.
