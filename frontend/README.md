# Frontend

Next.js app for Stock Picker (port **18730** in dev).

- **Start:** from repo root run `./scripts/dev-up.sh`, or `npm run dev` in this folder.
- **Docs:** see the main [README](../README.md) and [docs/](../docs/).

Dev uses Webpack (`npm run dev`), not Turbopack. If the dev server misbehaves: `./scripts/dev-down.sh` then `rm -rf .next` and restart.
